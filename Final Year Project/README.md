# Random Forest based Machine Learning Approach for Brain Tumor Analysis

This project focuses on brain tumor analysis using a machine learning approach. It was developed as a final year academic project to demonstrate how machine learning techniques can be applied to healthcare data.

## Description
A Random Forest model is used to analyze patient data and predict the presence of a brain tumor based on structured clinical features such as tumor size, age, and symptoms. The project includes data preprocessing, model training, evaluation, and visualization.

## Features
- Data preprocessing and cleaning
- Feature encoding and preparation
- Model training using Random Forest
- Prediction and evaluation
- Feature importance visualization using Matplotlib

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Result
- Achieved approximately 95% accuracy
- High recall for tumor prediction (important for medical analysis)
