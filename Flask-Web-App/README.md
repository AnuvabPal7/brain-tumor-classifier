# 📝 Notes Management Web Application

## 📌 Overview  
This is a simple web application built using Flask that allows users to create, manage, and organize their notes efficiently. It demonstrates core web development concepts, including backend development, routing, and database integration.

The project implements a basic full-stack CRUD system where users can add, view, edit, and delete notes with persistent storage.

## ✨ Features  
- Add new notes  
- View existing notes  
- Edit notes  
- Delete notes  
- Persistent storage using SQLite database  

## 🛠️ Technologies Used  
- Python  
- Flask  
- HTML  
- CSS  
- SQLite  

## ⚙️ Project Structure  
- Flask used as backend framework  
- HTML templates used for frontend rendering  
- SQLite database used for storing notes  
- CRUD operations implemented using Flask routes  

## ▶️ How to Run the Project  

To run this project use VS Code or any Python IDE.

### 1. Install Flask  
pip install flask

### 2. Open the project folder in your IDE  

### 3. Run the application  
python main.py

### 4. Open your browser  
http://127.0.0.1:5000

## ⚙️ Description  
This application is built using Flask as the backend framework and SQLite as the database for storing notes. It supports full CRUD (Create, Read, Update, Delete) operations, allowing users to manage notes dynamically.

The project demonstrates how a backend system interacts with a frontend interface to process user requests and manage data efficiently.

## 💡 Key Learnings

- Template rendering using HTML  
- Form handling in web applications  
- Database integration using SQLite  
- Full-stack application flow (frontend + backend interaction)
