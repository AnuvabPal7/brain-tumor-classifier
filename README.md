# 📘 NoteMaster Repository

This repository contains multiple projects developed during my B.Tech in Computer Science and Engineering, focusing on **Machine Learning, Web Development, and Applied AI systems**.

---

## 📂 Projects Included

### 🧠 1. Random Forest based Machine Learning Approach for Brain Tumor Analysis

A machine learning project that predicts the presence of brain tumors using structured clinical data.

**Key Features:**

* Data preprocessing and feature engineering
* Random Forest model for classification
* Model evaluation using accuracy, precision, recall, F1-score
* Feature importance visualization

**Tech Stack:** Python, Pandas, NumPy, Scikit-learn, Matplotlib

**Result:** Achieved ~95% accuracy with high recall for tumor detection

---

### 📝 2. Notes Management Web Application

A full-stack web application built using Flask that allows users to create, manage, and organize notes efficiently.

**Key Features:**

* Create, read, update, delete notes (CRUD operations)
* Backend routing using Flask
* Persistent storage using SQLite database
* Dynamic interaction between frontend and backend

**Tech Stack:** Python, Flask, HTML, CSS, SQLite

---

### 🤖 3. RAG-based Document Question Answering System

A fully offline **Retrieval-Augmented Generation (RAG)** system that answers questions from documents using a local AI model.

**Key Features:**

* Fully offline AI chatbot (no API keys required)
* Local LLM using Ollama (LLaMA3 / Mistral)
* Document-based question answering
* Fast semantic search using vector database (ChromaDB)
* HuggingFace embeddings for text representation

**Tech Stack:** Python, LangChain, ChromaDB, HuggingFace, Ollama
